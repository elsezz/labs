## Опис роботи

Програма реалізує інтерактивну візуалізацію гармонічного сигналу виду `y(t) = A · sin(ωt + φ)` із накладеним шумом та цифровою фільтрацією на базі `matplotlib.widgets`.

**Що реалізовано:**
- Функція `harmonic_with_noise` — генерує гармоніку з нормально розподіленим шумом
- Слайдери для керування амплітудою, частотою, фазою, параметрами шуму та частотою зрізу фільтра
- Чекбокс для перемикання відображення шуму
- Кнопка **Reset** — повертає всі параметри до початкових значень
- Фільтрація сигналу фільтром Баттерворта низьких частот (`scipy.signal.butter` + `filtfilt`)
- Два графіки: зашумлений сигнал та порівняння відфільтрованого з еталонним

> Шум генерується заново лише при зміні його параметрів. При зміні параметрів гармоніки шум залишається незмінним.

## Інструкція із запуску

1. Відкрийте термінал у кореневій директорії проєкту (наприклад, `lab4`).

2. Створіть віртуальне середовище:
   ```bash
   python -m venv venv
   ```

3. Активуйте віртуальне середовище:
   * **Windows:** `venv\Scripts\activate`
   * **Linux / macOS:** `source venv/bin/activate`

4. Встановіть необхідні залежності з файлу `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```

5. Запустіть файл:
   ```bash
   python lab4.py
   ```
