## Опис роботи

Лабораторна робота присвячена візуалізації багатовимірних даних, розвідувальному аналізу (EDA) та пошуку прихованих закономірностей з використанням бібліотек Matplotlib та Seaborn. Лабороторна робота базується на класичному наборі даних Automobile Dataset (UCI Machine Learning Repository).

Робота включає наступні етапи:
* **Завантаження та очищення даних:** Автоматизоване отримання даних, обробка пропущених значень (NaN) та перетворення типів даних для подальшого аналізу.
* **Базова візуалізація:** Побудова точкових графіків (Scatterplot) із лініями тренду для дослідження залежності ціни автомобіля від потужності двигуна, а також гістограм із заданими ціновими діапазонами.
* **Багатовимірна візуалізація:** Створення складних графіків для аналізу понад двох змінних одночасно, зокрема матриці попарних залежностей (Pairplot) та графіка паралельних координат (Parallel Coordinates).
* **Статистичні графіки:** Дослідження розподілу цін за типом кузова (Boxplot), виведення середніх значень за брендами (Barplot) та побудова теплової карти (Heatmap) для аналізу кореляції між усіма числовими атрибутами датасету.

## Вимоги до системи

* **Версія Python:** 3.10 або вище
* **Пакетний менеджер:** pip

## Інструкція із запуску

1. Відкрийте термінал у кореневій директорії проєкту (наприклад, `lab3`).

2. Створіть віртуальне середовище:
   ```bash
   python -m venv venv
   ```

3. Активуйте віртуальне середовище:
   * **Windows:** `venv\Scripts\activate`
   * **Linux / macOS:** `source venv/bin/activate`

4. Встановіть необхідні залежності з файлу `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```

5. Запустіть середовище Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
