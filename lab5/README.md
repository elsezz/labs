## Опис роботи

Програма реалізує інтерактивний веб-додаток на базі фреймворку `Streamlit` для зручного фільтрування, аналізу та візуалізації даних вегетаційних індексів (VCI, TCI, VHI) по областях України (використовуються дані з Лабораторної роботи 2).

**Що реалізовано:**
* Функція load_data — завантажує та об'єднує CSV-файли з даними VHI по областях, виконує очищення та нормалізацію даних, відображає числові ідентифікатори NOAA на назви українських регіонів
* Dropdown-списки для вибору індексу (VCI, TCI, VHI) та області аналізу
* Слайдери для задання діапазону тижнів та років, за якими відбираються дані
* Два чекбокси для сортування таблиці за зростанням або спаданням обраного індексу з обробкою конфліктного стану (обидва увімкнені одночасно)
* Кнопка Reset — повертає всі фільтри до початкових значень через st.session_state
* Три вкладки для відображення результатів: таблиця відфільтрованих даних, графік часового ряду та графік порівняння областей
* Графік часового ряду (plotly.express.line) — відображає динаміку обраного індексу для вибраної області за заданий період
* Графік порівняння (plotly.express.bar) — показує середнє значення індексу по всіх областях за вказаний інтервал, де обрана область виділена кольором
* Інтерфейс розділено на дві колонки: панель керування зліва та область результатів справа

## Вимоги до системи

**Версія Python:** 3.10 або вище

## Інструкція із запуску

1. Відкрийте термінал у кореневій директорії проєкту (наприклад, `lab5`).

2. Створіть віртуальне середовище:
   ```bash
   python -m venv venv
   
3. Активуйте віртуальне середовище:
   * **Windows:** `venv\Scripts\activate`
   * **Linux / macOS:** `source venv/bin/activate`

4. Встановіть необхідні залежності з файлу `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```
5. Запустіть веб-додаток за допомогою Streamlit:
   ```bash
   streamlit run lab5.py
   ```

## Примітки щодо роботи програми
* Додаток автоматично зчитує та об'єднує всі .csv файли з папки vhi_data при запуску. Перед запуском треба переконатися, що ця папка знаходиться в одній директорії зі скриптом lab5.py.
* Оскільки це веб-додаток, після виконання команди streamlit run програма автоматично відкриє інтерфейс у браузері за адресою http://localhost:8501.
