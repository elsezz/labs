## Опис роботи
Ця програма розширює функціонал попередньої лабораторної роботи, додаючи детекцію облич у реальному часі за допомогою нейронної мережі ResNet-10 SSD через модуль `cv::dnn`. Для забезпечення плавності відео (30–60 FPS) інференс нейромережі винесено в окремий фоновий потік за допомогою `std::thread` та `std::mutex`.

## Функціонал та керування
Під час роботи програми ви можете перемикати режими обробки та взаємодіяти з відеорядом:

* **Клавіатура (Режими обробки):**
  * `1` — Звичайний режим (Normal).
  * `2` — Інверсія кольорів (Invert).
  * `3` — Розмиття Гауса (Gaussian Blur).
  * `4` — Фільтр Canny (Виділення контурів).
  * `5` — Детекція облич (Face Detection).
  * `Q` або `ESC` — Завершення роботи програми та закриття камери.

* **Миша:**
  * **Слайдер "Brightness":** Регулювання яскравості кадру (від -50 до +50 від базового значення).

## Архітектура проєкту

| Клас | Відповідальність |
|------|-----------------|
| `CameraProvider` | Інкапсулює `cv::VideoCapture`, відкриває `/dev/video0` через V4L2 |
| `Display` | Керує вікном OpenCV, трекбаром яскравості, викликає `imshow` |
| `KeyProcessor` | Обробляє натискання клавіш, зберігає поточний `ProcessingMode` |
| `FrameProcessor` | Застосовує ефекти до кадру, малює рамки облич, FPS та назву режиму |
| `FaceDetector` | **Новий.** Асинхронна детекція облич у фоновому потоці (`std::thread`) |

### Логіка багатопотоковості (`FaceDetector`)

```
Main Thread                        Worker Thread
──────────────────────────────     ──────────────────────────────
getFrame() → frame                 loop:
submitFrame(frame) ──[mutex]──►      якщо є новий кадр:
                                       запускає net.forward()
getDetections()    ◄─[mutex]──        зберігає результат ──[mutex]
drawRects + imshow
waitKey(1)
```

Головний потік **ніколи не блокується** на інференсі — FPS залишається стабільним, рамка оновлюється асинхронно.

## Вимоги до операційної системи та заліза
* **Операційна система:** Linux (протестовано на Kali Linux).
* **Апаратне забезпечення:**
  * Процесор: 1.5 GHz або швидший.
  * Оперативна пам'ять: від 2 GB.
  * Наявність підключеної веб-камери (пристрій `/dev/video0` або аналогічний).
* **Залежності для збірки:**
  * Компілятор з підтримкою C++17 (GCC/G++).
  * CMake (версія 3.10+).
  * Make.
  * Бібліотека OpenCV (версія 4.x, з модулем `dnn`).

## Інструкція для автоматизованої збірки та запуску

Відкрийте термінал та клонуйте репозиторій. Після цього перейдіть в робочу директорію:
```bash
git clone https://github.com/elsezz/labs.git
cd labs/lab7/Lab7_OpenCV
```

Надання права доступу скриптам (за потреби):
```bash
chmod +x preinstall.sh build.sh run.sh
```

Для встановлення залежностей скрипт автоматично встановить CMake, компілятори, бібліотеку OpenCV та завантажить файли нейронної мережі (потребує прав адміністратора):
```bash
sudo ./preinstall.sh
```

Збірка проєкту — скрипт створить директорію `build`, запустить CMake та скомпілює вихідний код:
```bash
./build.sh
```

Запуск програми (може знадобитися `sudo`, якщо система вимагає прав root для доступу до `/dev/video0`):
```bash
./run.sh
# або
sudo ./run.sh
```
